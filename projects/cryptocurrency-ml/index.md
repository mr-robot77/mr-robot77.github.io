## Overview

This project applies advanced machine learning techniques to cryptocurrency market data, focusing on pattern recognition, clustering analysis, and predictive modeling. The goal is to identify trading opportunities and understand cryptocurrency behavior through data-driven approaches.

By leveraging machine learning algorithms, this analysis provides actionable insights for traders and investors looking to make informed decisions in the volatile cryptocurrency market.

## Methodology

### 1. Data Preparation
- Collected historical cryptocurrency price data
- Feature engineering for technical indicators
- Data normalization and scaling for ML models

### 2. Trading Signal Generation
- Implemented Exponential Moving Average (EMA) calculations
- Developed buy/sell signal logic based on EMA crossovers
- Backtested signals against historical data

### 3. Clustering Analysis
- Applied K-Means algorithm to group similar cryptocurrencies
- Analyzed cluster characteristics and patterns
- Identified coin groups with similar market behavior

### 4. Predictive Modeling
- Built regression models for price forecasting
- Feature selection for optimal model performance
- Cross-validation for model reliability

## Machine Learning Models

### K-Means Clustering
- **Purpose:** Group cryptocurrencies by behavior
- **Features Used:** Price volatility, trading volume, market cap, price trends
- **Results:** Successfully identified 3-5 distinct coin groups

### EMA-Based Signal Generation
- **Purpose:** Generate automated trading signals
- **Approach:** Multiple EMA periods for crossover detection
- **Outcome:** Identified high-probability entry and exit points

### Price Prediction Models
- **Algorithms:** Linear Regression, Random Forest, LSTM
- **Target:** Short-term price movements (1-7 days)
- **Evaluation:** RMSE, MAE, R² score metrics
